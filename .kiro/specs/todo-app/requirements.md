# Requirements Document

## Project Description (Input)
個人のタスク管理において、シンプルかつ使いやすいWebアプリがない。期限日・優先度の管理が煩雑で、タスクの見落としが発生する。

タスクの追加・編集・完了・削除ができ、期限日と優先度（高/中/低）を設定・確認でき、フィルタリング・ソートでタスクを整理できるSPA（React 19 + TypeScript + Vite）を構築する。データはlocalStorageに永続化し、バックエンドやユーザー認証は不要。モダンで洗練されたUIで快適に操作できることを目指す。
